# 🛡️ ArtGuard AI

ArtGuard is an AI-powered web application built with Flask that helps protect and manage digital artwork by adding security, upload handling, and AI-based processing features.

---

## 🚀 Features

- 📤 Upload images securely  
- 🔐 Basic protection for uploaded content  
- 🧠 AI-powered processing (if integrated in app.py)  
- 🌐 Web-based interface using Flask  
- 🗂️ Organized file handling system  

---

## 🛠️ Tech Stack

- Python 🐍  
- Flask 🌐  
- HTML / CSS 🎨  
- SQLite (local database)  
- Git & GitHub
